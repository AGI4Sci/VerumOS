## ⚠️ 注意事项

### API 限速

| API | 限速 | 处理方式 |
|-----|------|---------|
| NCBI | 3次/秒 | ✅ 系统自动延时 |
| PubChem | 无明确限制 | 建议间隔 0.5秒 |
| ChEMBL | 无明确限制 | 建议缓存结果 |
| UniProt | 无明确限制 | 建议使用 ETag |

### 数据缓存

- 相同查询会自动缓存结果
- 缓存有效期：24小时
- 可手动刷新数据

### 错误处理

如果查询失败，系统会：
1. 显示错误信息
2. 提供替代方案
3. 建议检查输入格式
