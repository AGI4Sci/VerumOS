## 需求文档模板

```markdown
# 需求文档标题

## 数据来源

| 文件 | 描述 |
|------|------|
| your_data.csv | 数据说明 |
| another_data.csv | 数据说明 |

## 目标

一句话描述你想要的结果。

## 输出要求

- 文件名：output.csv
```

---

## 字段说明

| 字段 | 是否必填 | 关键词 | 说明 |
|------|---------|--------|------|
| **数据来源** | ✅ 必填 | `## 数据` | 标题必须是"数据来源"、"数据源"或"数据" |
| 表格文件列 | ✅ 必填 | 文件扩展名 | 必须以 `.csv` `.tsv` `.xlsx` `.xls` 结尾 |
| **目标** | ✅ 必填 | `## 目标` | 标题必须是"目标"或"处理目标" |
| 输出要求 | ❌ 可选 | - | 输出文件名、格式等 |

---

## 最简示例

```markdown
# 数据整合

## 数据来源

| 文件 | 描述 |
|------|------|
| count_matrix.csv | 表达矩阵 |
| cell_metadata.csv | 细胞信息 |
| gene_annotation.csv | 基因注释 |

## 目标

整合为长格式宽表，输出 integrated_data.csv
```
